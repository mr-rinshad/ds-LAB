About Types of List in Html

1) Ordered list

In HTML, an ordered list is created using the ol tag and it is used to display items in a specific sequence. Each item inside the ordered list is written using the li tag. By default, the list is numbered with digits (1, 2, 3 …), but the type attribute allows us to change the style of numbering.
The different types of marks that can be used are:

  1. numbers (type="1")
  2. uppercase letters (type="A")
  3. lowercase letters (type="a")
  4. uppercase Roman numerals (type="I")
  5. lowercase Roman numerals (type="i")

2) Un-ordered list

In HTML, an unordered list is created using the ul tag and it is used to display items without any particular order. Each item inside the list is written using the li tag. By default, the list items are marked with solid bullets, but the type attribute allows us to change the bullet style.
The different types of marks that can be used are:

  • disc (type="disc")
  • circle (type="circle")
  • square (type="square")
  • none (type="none")

3) Definition list

In HTML, a definition list is created using the dl tag and it is used to display terms and their descriptions. Inside a definition list, the dt tag is used to define the term, and the dd tag is used to provide the definition or description of that term.This type of list is often used for glossaries, question–answer sections, or to describe items with details. Unlike ordered or unordered lists, a definition list presents data in a term–description format rather than with bullets or numbers.
The tags and their defintion listed using dl are geven below:


<dl>
It defines the start and end of a definition list in HTML.
<dt>
It is used to define a term or name inside a definition list.
<dd>
It provides the description or definition of the term specified by <dt>.
